OdooPortable
==============

A portable Odoo environment with included Python 2.7, PostgreSQL, wkhtmltopdf and node.js. It will run out of the box!

Download ready to use files
-----
You can download prebuild version on http://www.cybesystems.com -> Downloads. I've compiled a PAF (PortableApps) and a 7z compressed version. 

PortableApps
-----
PortableApps (http://portableapps.com/) Tool is supported.

Compile
-----

If you want to compile CygwinPortable by yourself you need Python 3.4, PyQT, psutil and pyDistribute (https://github.com/CybeSystems/PyDistribute) - Run pyDistribute.py and it should work :-)
